## bobo cli

Bobo is a quality of life improvement for building web apps and services.

### Synopsis

Bobo is a convenient tool to bootstrap your web apps and services. Write your first routes within 10 min.

```
bobo [flags]
```

### Options

```
  -h, --help   help for bobo
```
